# LAB 04 – Fundamentals of Computer Programming (Functions)(C++)

This project contains solutions for LAB 04 – Function practice tasks.

## Implemented Tasks
- sum3(a, b, c) – sum of three numbers  
- print_stars_advanced(s, symbol) – prints symbols in brackets  
- factorial(n) – recursive factorial  
- estimate_pi(shots) – Monte Carlo approximation of π  
- minimum(a, b) – inline function returning the smaller value  


## How to Run

g++ "task 4.cpp" -o task4
./task4
