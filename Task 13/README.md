# Task 13: Smart Pointers and Custom C-String Functions

## Description
This program demonstrates the use of  C++ smart pointers and custom C-string operations:

## Tasks:
_Create and Use unique_ptr
_Transfer Ownership with unique_ptr
_Create Dynamic Array with unique_ptr
_Shared Ownership with shared_ptr
_Custom strlen Implementation
_Custom strcpy Implementation
_Custom strcat Implementation
_Count Words in C-String

Each task is implemented as a separate function and tested in the main function.

## How to run

 g++ task13.cpp -o task13
   ./task13
