Digital Gradebook

Description

Digital Gradebook is a student gradebook simulation written in C++. The application allows managing users and students, creating grade placeholders, and assigning grades. Students can log in to view their grades , while the admin controls grade management. All data is stored in text files.
Files

* digital.cpp – main source code
* students.txt – student data
* subjects.txt – subject list
* smalltest.txt – small test data
* largetest.txt – large test data
How to compile

g++ digital.cpp -o digital
How to run

./digital
